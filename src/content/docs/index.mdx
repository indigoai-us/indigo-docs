Indigo HQ is the operating system for AI workers — a shared context layer over Claude Code, Cursor, and Codex that syncs knowledge, skills, policies, and capabilities across your team. It scales from a solo builder to an enterprise.

HQ ships as a **split ecosystem** of independently versioned products rather than one combined repository. This site is the comprehensive user + technical reference for all of them. For how they compose, see the [Ecosystem & Repo Map](/hq/architecture/2-ecosystem/).

## Get started

```bash
npx create-hq
cd hq && claude
/setup
```
